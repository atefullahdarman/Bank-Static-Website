📖 Description
This project is a simple Bank Management System 
designed as a static website using HTML and CSS.
It demonstrates the basic structure and functionalities 
of a banking system without real backend processing.
🎯 Objective
To understand the basic concept of a banking system
To practice HTML and CSS skills
To design a user-friendly banking interface
⚙️ Key Features
Account management (UI only)
Deposit, withdraw, and transfer (design level)
Loan request section
Transaction and report display
Simple and clean user interface
🛠️ Technologies Used
HTML
CSS
🚫 Limitations
No backend functionality
No database integration
No real authentication system
All data is static (for design/demo only)
▶️ How to Run
Download or clone the project
Open the project folder
Run index.html in any web browser
📌 Conclusion
This project is a basic representation of a banking system
for learning purposes and can be enhanced in the future by adding 
JavaScript and backend technologies.
